Welcome to the greatest Trivia Game of all time

How to play:
  1. Make an account or log into a pre-existing account.
  2. Click the 'plus' icon in the nav bar to go to our game setup page.
  3. On the create game page you'll decide the game factors:
  4. Total players who will be playing the game
  5. Please note solo players cannot win or lose a game, but can each points for questions right
  6. Category
  7. Difficulty
  8. Type of Question (true/false or multiple choice)
  9. The other players joining you (if multi-player)

The game will alternate which players answer which questions, where at the end of the game the scores are tallied up.

The game will calculate each player's statistics to determine where their weaknesses and strengths are, and uploading their overall stats through previous games online.
